# 3.3  工作台

工作台是每个成员个人办公和工作的主要界面，并且支持每个人个性化的信息展示。  


## 自定义工作台

如果需要调整工作台的小部件，可以点击【自定义工作台】进行配置。  
![](http://cd8.okayapi.com/yesyesapi_20210531160459_e5ed206b537675356a9b525409fbd712.png)  

页面布局分为左侧布局、右侧布局。  

 + **左侧布局：**，以信息展示为主，着重与个人办公密切相关的具体信息，例如：项目/需求/任务/问题/算力。  
 + **右侧布局：**，以信息流和实时统计数据为主，重点提供辅助和支撑信息，例如：我的关注/我的消息/统计。  

同一区域内，可以上下拖动排序，调整自己想看的顺序。同时，可以根据自己的岗位角色，勾选感兴趣的小部件。  

例如，YesDev内置了以下常用的岗位角色：  
 + 我是研发  
 + 我是产品  
 + 我是测试  
 + 我是负责人  
 + 我是需求方  
 
## 11个工作台小部件

工作台小部件是可扩展、可个性化定制的，后续会逐步丰富和升级，目前提供的小部件有：  

### 左侧小部件

 + 我的任务排期
 + 我的待办任务
 + 我指派的任务
 + 我提的需求
 + 我的需求
 + 我的问题  
 + 我创建的问题
 + 我的项目
 + 工作台算力

下面分别简单介绍。  

**我的任务排期**，按日历划分的个人任务，以每一天为时间范围，显示个人TODO、DOING、DONE的任务，方便个人按时间线规划和回顾自己的任务。  
![](http://cd8.okayapi.com/yesyesapi_20210531161748_8dc8857fdfe73d75feb5fc17c23c4a6d.png)  

**我的待办任务**，显示个人需要全部待完成的任务，包括TODO、DOING，以及最近完成的任务，方便个人汇总自己的工作任务。  
![](http://cd8.okayapi.com/yesyesapi_20210531161934_552307757914096c0f4c5de1ceec3f08.png)  

**我指派的任务**，可以快速查看我指派给他人的任务，以便了解分配任务的完成情况。如果你是上级领导或需求方，此功能将会很大的帮助。  
![](http://cd8.okayapi.com/yesyesapi_20210531162345_1e2ccd042d07034b57ca074a4dbeee80.png) 

**我的需求**，我负责或需要完成的需求，点击可进入需求详情页，查看更具体和细致的需求详情、需求附件和需求PRD等信息。  
![](http://cd8.okayapi.com/yesyesapi_20210531162707_efbc6702102863f22bedd6c1489612c4.png)  

**我提的需求**，由我提交的需求，例如产品经理提交的产品迭代需求，又如技术人员创建的技术优化需求。  
![](http://cd8.okayapi.com/yesyesapi_20210531162850_3622ffb1f640b27b7f7b37edcf4b223a.png)    

**我的问题**，指派给我的问题、Bug、缺陷、用户反馈需求、设计问题等。即需要我解决或确认的问题。  
![]()http://cd8.okayapi.com/yesyesapi_20210531163046_a5e7b7422ffbb653cdd5a739b223045b.png  

**我创建的问题**，由我创建和提交的问题、Bug、缺陷、用户反馈需求、设计问题等。当指派人解决后，需要创建人在验收后把问题状态改为【已关闭】。  
![](http://cd8.okayapi.com/yesyesapi_20210531163148_16a270a8b9e50a4c551bfe425da58278.png)  

**我的项目**，由我负责、主导或统筹的项目。  
![](http://cd8.okayapi.com/yesyesapi_20210531163148_16a270a8b9e50a4c551bfe425da58278.png)  

**工作台算力**，根据近期任务、问题、项目等数据，计算个人的工作算力，分别分析本周和本月的工作产能、质量系数、影响力系数。  
![](http://cd8.okayapi.com/yesyesapi_20210531163338_b8b6f30064d7144ba3a5e8a865e7da16.png)  



### 右侧小部件

右侧小部件主要有：  
 + 我的关注
 + 我的消息
 + 工作组任务统计

**我的关注**，显示我关注的项目、需求、问题，点击可进入对应的详情页。如果关注的内容已划删除线，表示可不用再关注，因为可能已完成或已关闭或已解决。  
![](http://cd8.okayapi.com/yesyesapi_20210601102857_d7dd229ac7338d05649fa148616016cd.png)    

**我的消息**，站内消息推送，主要和项目、需求、问题、任务有关。新消息会有相应的标注和提示。  
![](http://cd8.okayapi.com/yesyesapi_20210601103031_6190630ecc70826b2232c2341964eff1.png)  

**工作组任务统计**，按工作组为分组，统计当前项目TODO、DOING和DONE的工时。  
![](http://cd8.okayapi.com/yesyesapi_20210601203505_6a9c305e4073451d5e06037064354318.png)
