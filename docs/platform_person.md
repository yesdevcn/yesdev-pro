# 3.9 个人空间

个人空间功能包括个人资料管理、修改密码和文件管理等。  


## 个人资料

在个人资料，可以查看和修改自己的头像、姓名、手机号和邮箱地址。  
![](http://cd8.okayapi.com/yesyesapi_20210601121804_0028e420fefff0d173b091c320dc26ae.png)  

> 温馨提示：手机号可用于快捷登录，一个手机号可以绑定多个企业。  

> 温馨提示：邮件地址未设置或设置不对，将无法接收实时的邮件通知。  

## 邮件通知订阅

个人可以选择订阅感兴趣的邮件通知主题。  

目前分为：  
 + 项目邮件通知
 + 需求邮件通知
 + 任务邮件通知
 + 问题邮件通知

![](http://cd8.okayapi.com/yesyesapi_20210601121957_f27d091ed5c488056e9e34a0b49d6fb0.png)  

如果取消了邮件通知，也不需要担心，因为在站内的推送消息也可以继续查看。

> 使用技巧：为方便整理邮件，可以在本地邮件软件为YesDev设置过滤器，自动分类。  

## 文件管理

在文件管理，可以按分类对文件进行查看、下载和管理。  


目前文件分为：  
 + 项目附件
 + 需求附件
 + 问题附件
 + PRD附件
 + 未关联的附件

![](http://cd8.okayapi.com/yesyesapi_20210601123736_8e9154c20a1fcc6f819d3650cef6ac58.png)  

## 素材库
在编辑器点击图片进行上传和管理时，可以调取个人的素材库。  

![](http://cd8.okayapi.com/yesyesapi_20210601124426_561ba8dc8335cc9ab3ffbf79b2aadc4c.jpg)
