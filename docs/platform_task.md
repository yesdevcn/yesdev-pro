# 3.6 任务协作

## 添加新任务
在顶部菜单，点击【添加】-【新任务】，或在其他界面进行任务添加，会显示添加任务的弹窗。  
![](http://cd8.okayapi.com/yesyesapi_20210531164747_89a8c314670291912fc37b465ad9e5d9.png)  


添加新任务时，可以设置以下信息：  
 + 任务标题（必填）
 + 指派给（必填，自己或他人）
 + 工时评估（单位：H，小时，保留一位小数）
 + 任务类型（其他/UI设计/产品原型/技术开发/测试/会议/编写文档/调研/沟通）
 + 关联项目
 + 关联需求
 + 任务描述
 + 完成时间（精确到天）
 + 任务状态（TODO/DOING/DONE）

添加新任务时，可以选择是否邮件通知负责人。  

## 任务饱和度提示
在为自己或他人添加新任务时，若指派人只有一个人，在选择计划完成时间时，日历插件将会及时显示当前任务指派人的任务饱和度，可用于快速评估和获取团队成员的工作负荷和饱和情况，进行合理安排和分配工作。  

![](http://cd8.okayapi.com/yesyesapi_20210531230118_b8f5bd5b63ad993957744d5b9f526991.png)  

> 温馨提示：红色表示工作已饱和，蓝色表示已安排工作但未饱和，白色表示无任何工作安排。  


## 任务顺延
如果你已经做了任务计划，突然需要插入紧急需求或任务时，可以使用任务顺延功能。  

当添加的任务工时大于4H，即大于半个工作日时，并且当前用户在大于本次任务计划完成时间仍然有未完成的任务时，将会出现任务顺延的提示。  
![](http://cd8.okayapi.com/yesyesapi_20210531224906_cd2dfa75576247110bc63d6bb2b3de55.png)  

鼠标移过可以看到任务顺延的明细。  
![](http://cd8.okayapi.com/yesyesapi_20210531225001_973fc9200d18188200a6ef444ff8e88f.png)  

如果确认需要顺延，可以勾选任务顺延，然后确定，就可以在添加新任务的同时，将原来计划内的任务进行顺延，不需要再一个个任务进行调整。  

## 任务看板

看板最初是丰田汽车公司提出来的，后来应用在敏捷开发中。YesDev的任务看板能在团队的项目协作过程中提供非常及时、有效和可量化的任务单元和信息。  

以下是YesDev的任务看板界面截图。  

![](http://cd8.okayapi.com/yesyesapi_20210610144706_587ef4ed09b55d3fbb7513c6c32345fe.png)  

### 我的任务看板
在软件开发过程中，首先需要了解的就是我自己的任务，以及每个任务的状态。因此，YesDev在设计上，默认显示的是我的任务，即把我当前需要TODO待办、DOING进行中和DONE最近完成的任务罗列出来。  

### 我部门的任务（正式组织架构）

在YesDev中的部门，是对应企业正式组织架构的一级/二级/三级部门，通过切换到我所在的部门，可以查看和我在同一个部门的同事，目前都有哪些任务。通常，对于上级和负责人，这种划分会更有价值。因为可以对团队的人员任务情况一目了解。

![](http://cd8.okayapi.com/yesyesapi_20210610144841_3ebd10ad51a0836e3ad11866a23e3626.png)  

### 工作组的任务看板

在平时工作中，很多项目是虚线的、临时的、以及需要不同部门的人员参与，这时可以通过工作组来灵活组队。然后，切换到对应的工作组，就可以查看当前工作组的任务情况，方便更加聚焦指定范围的项目都有哪些任务和工作，以及完成情况如何。  

![](http://cd8.okayapi.com/yesyesapi_20210610144946_bc2510b7a848fb0489118ac21377315d.png)  

### 选择你喜欢的分类方式

默认情况，是没有分类方式的，但你可以选择以下分类方式：

 + **按任务类型**  
 你可以清晰看到每个专业和每个类型的任务的分布情况。  

 + **按人员**  
 可以按人员分组，显示每个人当前的工时，以及各个任务的状态。  

 + **按项目**  
 会优先显示快完成的项目，方便团队重点关注剩下“最后一公里”的项目。通常，项目负责人或项目经理会更关注这个维度。

 + **按需求**  
 优先显示最老的需求，提醒团队不能忽略任何一个需求，特别是“老”需求，迟迟未上线的需求。通常，产品经理会更关注需求维度的任务完成情况。  

![](http://cd8.okayapi.com/yesyesapi_20210610145118_82a1ec483ab9808241cb91e876566c41.png)  

最后，你可以在任务看板，快速完成信息的更新和录入，例如添加新任务、拖动修改任务状态、点击查看任务详情。以及按项目分组和按需求分组时查看相应的项目详情和需求详情。 

## 工作排期
工作排期，可以按周为视图，统计和显示部门或工作组内每位成员的工作任务计划和完成情况。  
![](http://cd8.okayapi.com/yesyesapi_20210531230325_547e6cc4a34a6d0d9940e65ca086ac3d.png)  

同时，可以导出Excel表格，例如用于每周的人员工时汇报和登记。  
![](http://cd8.okayapi.com/yesyesapi_20210531230532_5fc3cc2420d27bbd118354b97fbe28d6.png)  

## 任务墙

若需要对比多个成员的个人任务情况，可以使用任务墙，例如在多个相同岗位或专业技术人员进行任务的对比，能更方便进行任务的合理的调配。  
你可以自由选择多个成员。  
![](http://cd8.okayapi.com/yesyesapi_20210531230800_e96c974169bf96ad000e1fca0102c3f9.png)  

选择后，会及时显示所选择的成员的未完成任务，可以对成员置左，或进行整理。  

![](http://cd8.okayapi.com/yesyesapi_20210531230626_09430df128dd9c0494705e78b3afff86.png)
