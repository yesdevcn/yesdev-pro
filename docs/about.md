# 关于我们

**专 业 · 创 新 · 品 质 · 服 务**  

YesDev是一款项目管理工具，由果创科技公司出品，为互联网企业提供项目管理协助系统。它是以敏捷开发和MVP理念设计的一套管理工具，旨在满足企业团队协作、产品项目、研发管理需求。  

创始人黄禅宗（dogstar）是PhalApi开源框架作者，团队核心成员来自唯品会、租租车、汇量等上市或知名企业，拥有丰富的互联网技术和软件开发经验。
