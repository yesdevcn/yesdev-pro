
## 1.2.7 任务查询

企业管理员可以对详细的任务和工时进行后台查询，类似对考勤的纪录进行查询以及导出。  

![](http://cd8.okayapi.com/yesyesapi_20210907152710_65028d09b85dc6278c3a6b574258c44d.png)    

在任务查询页面，可以查询指定成员或多个成员，或者指定部门，在某个时间段内的任务纪录。  

任务列表展示的信息包括：  
 + 姓名
 + 任务标题
 + 工时评估(H)
 + 任务状态
 + 完成时间
 + 任务类型
 + 创建时间
 + 推测延期天数

搜索后，也可以导出全部任务纪录。 

## 1.2.8 成员工作分析

企业管理员可以指定单个成员以及工作时间范围，进行成员工作分析。  

![](http://cd8.okayapi.com/yesyesapi_20210907152749_ca387fbc506c85bbaa8dd36c1205db1a.png)  

可以查看成员个人的以下指标：  

 + 项目统计：主导项目、参与项目、影响力 
 + 工作效率：个人任务、任务总工时、平均延期  
 + 工作产出：完成需求、负责需求  
 + 质量：已修复、负责问题、每日缺陷  
 + 贡献统计与对比：对比个人、部门和公司  
 + 任务分析：任务延期统计与分布
 + 缺陷分析：解决Bug的时长统计与分布
