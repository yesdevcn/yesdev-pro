# 3.4 需求流转

需求是由产品经理或用户提出的需求、故事点或问题。  


## 提需求

当需要提需求时，可以通过顶部的菜单【添加】-【新需求】，创建新的需求。  
![](http://cd8.okayapi.com/yesyesapi_20210531172436_bf820e10cd95ab5e9e742ee5ec80095e.png)  

提需求时，可以设置以下需求信息：  
 + 需求名称
 + 绑定PRD（可上传PRD到YesDev并进行关联）
 + 优先级（紧急/高/中/低）
 + 评审状态（待评审/已评审/不需要评审）
 + 需求状态（需求中/设计中/产品中/研发中/测试中/已上线/已完成/挂起）
 + 产品线（可自行创建产品线并绑定）
 + 负责人
 + 需求方
 + 计划上线时间（精确到天）
 + 需求内容（支持富文本编辑）

此外，在创建新需求时，可以选择是否邮件通知负责人。 

## 需求池

新需求创建后，可以在需求池查看和编辑。

![](http://cd8.okayapi.com/yesyesapi_20210531173522_23ca35ab10904652d3abfeb08f8ae115.png)  

## 关联项目
需求最多可关联一个项目，在创建新项目或编辑项目时，可以把需求和项目进行关联。  
![](http://cd8.okayapi.com/yesyesapi_20210531173718_48041ada360dce03cdc93863d4d1fd13.png)  

需求关联项目后，可以在需求详情页、需求池和项目详情页等界面查看。  
![](http://cd8.okayapi.com/yesyesapi_20210531173924_56ec90fe9adf8c8753ceffc0ffaeb5fa.png)  


## 需求排期

当需求在线下进行需求评审，并且在线上进行关联项目后，结合产品经理和需求方的上线期望时间，和研发团队的实际排期，可以得到需求排期。  
需求排期为产品经理、需求方和研发团队，提供了每个需求的日历视图，方便了解和查看每个产品业务线的需求迭代计划和进度。  

![](http://cd8.okayapi.com/yesyesapi_20210531174355_262ff0605719aadb831bd7a94ac821a6.png)  

你也可以导出需求排期Excel表格，进行归档、分享和调整。  
![](http://cd8.okayapi.com/yesyesapi_20210531174530_78440ac3380079f3e5b7fd932f88a46f.png)  


## 需求进度
当研发团队进行任务工时的评估和登记，并且把任务和需求关联后，就可以得到单个需求的进度统计情况。  

![](http://cd8.okayapi.com/yesyesapi_20210531180316_3cb746d0f238ff03c0d73cfc6cb42374.png)  


## PRD原型管理

### 上传PRD原型

通过【PRD原型管理】-【上传PRD】，可以把本地Axure等原型设计软件发布的HTML压缩包上传到YesDev进行托管和协作。 
![](http://cd8.okayapi.com/yesyesapi_20210531180427_9f1079a763d9ec4fe50226c2c8025db8.png）  

上传成功后，可以在【PRD原型管理】进行查看和编辑。还可以复制分享链接给客户、团队或需求方。   
![](http://cd8.okayapi.com/yesyesapi_20210531180649_33bcbd748ce18781ca18eb626feb4e8d.png)  


## PRD在线演示

PRD原型管理主要是为PRD提供者即产品经理使用的，而PRD在线演示则主要是为使用、访问和查看PRD的人员使用的，例如客户、研发团队。  
![](http://cd8.okayapi.com/yesyesapi_20210531180812_818678a9f2415f5e9a1f4060934ac092.png)  

PRD在线演示，根据TODO\DOING\DONE分为3个区域，可以拖动，点击可直接在线查看PRD。  
例如以下PRD演示示例。

## 产品业务线

产品业务线，可用于规划企业的产品和业务，针对有形的产品或无形的服务进行组织和管理。  
设定的产品业务线，将可以用于关联需求、关联问题、关联PRD等。  

以下是产品业务线的维护页面，可以添加、删除、编辑产品线，以及进行产品线的版本管理。  
![](http://cd8.okayapi.com/yesyesapi_20210531181305_c941e89caf6b5df731b2746f3ff86148.png)  

比如，产品线在需求池的应用。  
![](http://cd8.okayapi.com/yesyesapi_20210531181347_35a0b7af20b4d13e34d4fd6cdc268de6.png)  

又强，产品线在问题管理的应用。
