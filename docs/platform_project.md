# 3.5 项目管理

YesDev的核心是项目管理与协作，可以把一次迭代或某个版本视为一个项目。在YesDev，项目是一个有限集合，包括在短时间或长时间内完成的需求、待执行的任务和要解决的问题。  

## 创建新项目

通过顶部菜单【添加】-【新项目】，或其他入口，可以创建新项目。  
![](http://cd8.okayapi.com/yesyesapi_20210531184901_b894c1e4b31038005b36904e70283517.png)  

创建新项目时，可以设置以下项目相关信息：  

 + 项目名称
 + 工作组（必填，**用于控制项目的访问权限**）
 + 项目时间（开始和结束时间，精确到天）
 + 所属产品
 + 项目负责人
 + 项目类型（迭代需求/新项目/重构项目/故障修复）
 + 项目需求（可关联多个未被关联的需求）
 + 项目描述（支持富文本在线编辑）
 + 项目状态（规划中/进行中/已完成/挂起）

创建新项目时，可以选择是否邮件通知项目负责人。  

## 项目管理

成功创建新项目的，可以在【项目管理】界面进行查看、管理和维护。  
![](http://cd8.okayapi.com/yesyesapi_20210531185239_64e87656557aea19837f2efa2196ad5c.png)  

需要注意的：**项目访问权限**和**项目状态**。如果在项目管理看不到某个项目，请核对权限和状态。  

 + **项目访问权限**  
 项目访问权限和工作组有关，只有项目所关联的工作组成员才有访问项目的权限。关于工作组的设置，后续会有相关说明，这里不再赘述。  

 + **项目状态**  
 项目状态分为：规划中/进行中/已完成/挂起，对于当前项目，包括的状态有：规划中和进行中。对于历史项目，包括的状态有：已完成和挂起。全部项目则包括全部状态的项目。  

> 温馨提示：如果未能查看某个项目，请核对项目访问权限和项目状态。  

## 当前项目

如前面所述，在YesDev中，对于当前项目，包括的状态有：规划中和进行中。即：  

**当前项目 = 规划中项目 + 进行中项目**  

YesDev核心是项目管理，而在项目管理中，最为关注的又是当前项目，即规划中和进行中的项目，以便减少项目管理和团队人员的关注点，默认把最宝贵的注意力和焦点聚集在当前项目。  

在很多功能和界面中，默认的展示和选择都是当前项目，使用过程中，请注意区分和进行必要的切换。  

例如项目管理的：当前项目/历史项目/全部项目。    
![](http://cd8.okayapi.com/yesyesapi_20210531190017_fb7ab41307f131b4df2ac5745cd759fc.png)  

又如项目进度时，仅显示当前项目：  
![](http://cd8.okayapi.com/yesyesapi_20210531190120_3b5bc4a06622c7bb2d5572e99ab09d52.png)  

> 温馨提示：使用过程中，请留意当前项目。

在项目详情，可以进行有效的单个项目协作，以及查看结构化的项目信息。 
![](http://cd8.okayapi.com/yesyesapi_20210531190252_8087ebd9078aaac9ccf630083c940bc2.png)  

当向下滚动查看更多项目信息，可以通过顶部的悬浮导航菜单快速定位感兴趣的信息。

## 项目进度

项目可以关联多个需求，项目也可以关联多个任务，同时需求也可以关联任务。关于项目、需求、任务，有以下几种关联关系：  

 + 项目 - 需求 - 任务
 + 项目 - 需求
 + 项目 - 任务
 + 需求 - 任务

简而言之，任务可以直接关联项目，也可以同时关联需求和项目。只有当需求未关联任何项目时，任务才会只关联需求而不关联项目。  

由此得知，项目的进度主要是通过任务的工时和完成状态来进行统计的。  

**项目进度 = DONE已完成任务总工时 / (TODO总工时+DOING总工时+DONE总工时) * 100 %**  

任务工时单位为H，即小时。  

你可以在项目管理、项目详情和项目统计，查看项目进度和相关的工时统计。  
![](http://cd8.okayapi.com/yesyesapi_20210531191245_0ba3b5d0808a58ba3a707d4b35500cd5.png)  

如果需要对当前项目进行进度的分析、查看和管理，可以使用【项目进度】。  
![](http://cd8.okayapi.com/yesyesapi_20210531191406_12d6c435f6f7efd503727235a59be515.png)  

## 项目统计

项目进度是针对单个项目的纵向管理，项目统计则是对多个项目横向的对比和汇总。  

![](http://cd8.okayapi.com/yesyesapi_20210531191604_d27f0196230e71610199edf7eccb2c21.png)  

同时，可以导出项目统计Excel表格，进行邮件沟通或归档。  
![](http://cd8.okayapi.com/yesyesapi_20210531191705_5972418f8399db02387f5305f8673949.png)  


## 项目甘特图

项目甘特图是一个非常有用的工具，它可以让你在更高的维度对项目集内的项目进行汇总和规划。  

![](http://cd8.okayapi.com/yesyesapi_20210531191832_2a416c6135015e9e38cb4e3d62f72e92.png)  


## 团队作战蓝图

作战蓝图，通过更加丰富的图表来呈现项目统计数据。  

目前包括：  
 + 实时项目雷达
 + 近期项目概况
 + 项目透视图-剩余工时
 + 项目总工时预告

![](http://cd8.okayapi.com/yesyesapi_20210601155811_6fa90af24000aec9e93e3ed8e8879c32.png)  

## 工作组

工作组可以根据团队的实际情况，和虚线和常规项目的需要，进行自由组团。  
![](http://cd8.okayapi.com/yesyesapi_20210531191931_2f661f0364862d50a534634f3b52c689.png)  

创建工作组时，可以设置谁可以看，可以选择：  
 + 公开（公司内可见）
 + 保护（工作组内可见）
 + 私密（仅自己可见）  

![](http://cd8.okayapi.com/yesyesapi_20210531192039_f283419fd49213e6b7de71f6b095a14b.png)  

创建工作组后，工作组的管理员可以对工作组成员进行添加、移除和调整。  
![](http://cd8.okayapi.com/yesyesapi_20210531192151_11671ea5961df406077b09ab7afe8fd1.png)  

对于新加入的团队成员，为了让其可以快速开展工作，请记得为他分配相应的工作组。
