# 4.3 程序员客栈雇佣项目协作  

> 温馨提示：使用前，请使用你的程序员客栈账号授权登录。  

## 4.3.1 我是企业
首先，需要在程序员客栈上，向有意向的程序员提交预约。  
![](http://cd8.okayapi.com/yesyesapi_20210720112510_0172b967d335811fe64c6b72081a3795.png)  

提交预约后，待程序员接受后，即可开始在线协作。  

### 访问YesDev项目协作

在程序员客栈的雇佣详情页面，可以点击【YesDev项目协作】访问。
![](http://cd8.okayapi.com/yesyesapi_20210720114006_21975f348ae4771503a4973a58903e58.png)  

进入YesDev后，可以和程序员一起在线协作任务、需求和问题等。  
![](http://cd8.okayapi.com/yesyesapi_20210720114326_dfa2ec129067ebaca4b7ba22fb8a95b4.png)  

在YesDev，你也可以通过【项目协作】-【项目管理】，查看更多的协作项目。  
![](http://cd8.okayapi.com/yesyesapi_20210720114434_3913c6fb211e23f21ad63cad316aa290.png)  


## 4.3.2 我是程序员

登录程序员客栈，进入【工作台】-【我的工作】-【雇佣项目】，对【待接单】的雇佣，进行确认。  
![](http://cd8.okayapi.com/yesyesapi_20210720112854_b91abc2d4fd1a37e40a5cad45f1fe8fa.png)   

确认没问题后，点击【接受】。  
![](http://cd8.okayapi.com/yesyesapi_20210720113004_10192f31da580f3a7f6816310baedc57.png)  

### 访问YesDev项目协作

在程序员客栈的项目页面，可以点击【YesDev项目协作】访问。  
![](http://cd8.okayapi.com/yesyesapi_20210720113103_201d68e3c8f5b663e22ca81c24d05e77.png)  

进入YesDev后，可以进行此雇佣项目的在线协作。  
![](http://cd8.okayapi.com/yesyesapi_20210720113355_f75dc039bde7ad521a58b3fdc75b3920.png)  

在YesDev，你也可以通过【项目协作】-【外部项目】，访问和查看其他外部协作项目。  
![](http://cd8.okayapi.com/yesyesapi_20210720205853_f287ea5268249167354cd2d037126b2b.png) 

### 添加任务

在YesDev的项目详情协作页面，可以根据工作需要，添加新任务，评估工时，填写计划完成的时间。  

![](http://cd8.okayapi.com/yesyesapi_20210720113723_c4880a098ed9ce453c4c9d63a659ac7b.png)   

### 完成任务

当完成自己添加的任务或者由客户指派的任务后，可以及时把任务状态修改为【DONE】。
