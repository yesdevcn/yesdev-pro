# 3.1 加入企业  

个人在使用YesDev，请先加入企业。个人不需要额外注册新企业，但需要加入已有的企业。  

个人加入企业的方式有多种，主要分为：企业管理员分配账号、邀请注册。  

## 企业管理员分配账号
可以由企业管理员直接在管理后台，添加新成员账号，或者导入账号。  

## 邀请注册  
另一种方式，可以由企业管理员或团队成员，分享邀请注册链接，进行自主注册和加入。此时，注册只会创建个人账号，不需要创建新团队。  

加入企业后，即可和团队一起在线协作项目啦！🎉  

## 切换团队或企业

如果您加入了多个团队或多个企业，可以在登录时进行团队的选择。
