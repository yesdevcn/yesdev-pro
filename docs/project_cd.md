## 输出物交付标准

从一个抽象的需求到最终具体上线的产品或系统，中间会涉及很多输出物或制品，例如：产品原型、源代码、UI设计稿、测试报告、压测报告、测试包等。  

这些中间的交付和输出，我们称之为输出物。输出物是由指定岗位的专业人员制作输出，然后交付流转交给下一个岗位的团队成员。例如，UI设计师设计好UI稿好交付给前端技术开发人员进行开发。至于何时需要何人输出何物，可由团队自己进行约定和规范。
