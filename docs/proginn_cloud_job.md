# 4.4 程序员客栈云端工和协作  

> 温馨提示：使用前，请使用你的程序员客栈账号授权登录。  

## 4.4.1 我是企业

### 发布远程工作
首先，企业客户需要在程序员客栈平台上，发起远程工作。  
![](http://cd8.okayapi.com/yesyesapi_20210805154342_48c657552f66536b77ed0ecf060136b3.png)  

发布后，等待平台审核和对接开发者。

### 开始协作

开发者接单后，登录进入YesDev即可和开发者、程序员一起远程协作。在【项目协作】-【项目管理】，可以查看最新创建的远程工作。
![](http://cd8.okayapi.com/yesyesapi_20210805160046_f4f1908dc162af2ce656ac92d9f7fa48.png)  

进入项目详情页，可以对远程工作过程中的需求、任务和问题进行登记、共享和协作。
![](http://cd8.okayapi.com/yesyesapi_20210805160135_82b91ad4b6b6c0b7a93a196216aa9689.png)  

## 4.4.2 我是程序员

### 确认接单
登录程序员客栈，可以看到新的云端工作。  
![](http://cd8.okayapi.com/yesyesapi_20210805154720_695ddf5599cf222dc1d85c774aee2cd6.png)  

对于平台匹配的云端工作，确认后可接单。  
![](http://cd8.okayapi.com/yesyesapi_20210805154843_603d31f0f322ab7bf71b84fe851edc3c.png)  

### 开始协作

成功接单后，登录进入YesDev即可和客户在线一起远程协作。在【项目协作】-【外部协作】，可以查看最新接单的远程工作。  
![](http://cd8.okayapi.com/yesyesapi_20210805155535_1dc6aec31edb980a57056c8eaa2053cc.png)  

进入项目详情页，可以对远程工作过程中的需求、任务和问题进行登记、共享和协作。  
![](http://cd8.okayapi.com/yesyesapi_20210805155756_5b669d5bc4aab2a8862c145c3c62323a.png)
