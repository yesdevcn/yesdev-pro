# 广益联募

## 一、平台介绍

![](./img/gyufc_logo.png)  

广益联募是民政部指定的首批全国慈善组织互联网公开募捐信息平台之一，以“做慈善事业的支持者”为宗旨，以“联结力量、增进公益”为使命，坚持“开放、透明、多元、融合”的理念，打造成为影响全国的“集善地”和全民慈善的便易平台。

广益联募规划“善捐、善助、善品、善行、善知”五大板块，其中“善捐”板块核心功能为慈善项目在线公开募捐；“善助”板块核心功能为个案求助；“善品”板块核心功能为慈善义卖；“善行”板块核心功能为慈善活动在线支持；“善知”板块核心功能为慈善行业能力支持。

广益联募由广州市广益联合募捐发展中心负责运营。

## 二、项目背景

2021年1月1日，广益联募平台正式启动广益联募重构计划，为慈善组织和爱心用户带来更舒适的使用体验。同时采用了YesDev进行在线项目管理与研发协作。

![](./img/gyufc-index.png)  


## 三、YesDev项目管理

YesDev是一站式企业研发管理、项目管理与协同办公平台，支持敏捷开发、DevOps、Scrum、硬件项目等多种迭代方式，能为企业管理者智能生成项目投入产出的数据模型，真正实现项目研发全流程数字化管理。拥有六大核心功能模块：项目管理、测试管理、效能度量、知识库文档、通知推送、企业管理等。
